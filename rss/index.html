关于本订阅

本订阅源自动抓取 桂林理工大学教务处（jwc.glut.edu.cn）发布的通知公告，按关键词筛选出与学生学业相关的重要通知，例如：选课、考试、毕业论文、报名、证书办理、缓考重修、实习答辩等。

同时通过负向关键词排除监考发车时间安排等次要信息，让通知更聚焦、更实用。系统每日自动运行，及时推送最新通知到你的阅读器。

使用建议

普通学生

如果你只是希望及时收到教务处的重要通知（选课、考试、毕业等），直接订阅本项目的 RSS 即可，无需任何额外操作。复制上方订阅地址到阅读器即可使用。

教师 / 进阶用户

如果你是教师，或是有其他筛选需求的学生，建议复刻（Fork）本项目，修改 keywords.json 中的关键词配置后自行部署到自己的 GitHub Pages，以获得更贴合自身需求的订阅内容。
